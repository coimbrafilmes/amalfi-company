## 📐 Configurações Padrão Canva pra Amazon

### Tamanho do canvas
- **Quadrado**: 2.000 × 2.000 px (recomendado)
- **Vertical**: 1.600 × 2.000 px (4:5 — aceito)

### Resolução de export
- PNG ou JPG, 300 DPI
- Qualidade alta (não comprimir excessivamente)

### Cores institucionais Amalfi (sugestão — owner pode ajustar)
- **Primária**: Dourado/Bronze `#B8860B` (combina com produtos linha luminária)
- **Secundária**: Branco `#FFFFFF`
- **Texto**: Cinza escuro `#1F1F1F` (não preto puro — fica menos agressivo)
- **Acento**: Verde sálvia `#7B8B6F` (cor sofisticada Amalfi-style)

> **Owner pode mudar essa paleta** — se você (Marco) souber que owner adotou outra paleta, atualize aqui na kb.

### Fontes (Canva grátis)
- **Headings**: Playfair Display (serifa elegante) OU Montserrat Bold (sans-serif moderna)
- **Body**: Open Sans Regular (legível)
- **Highlights**: Montserrat Black (impacto)

### Briefing #1 — Infográfico de Features (imagem #3)

**Objetivo**: comunicar 4 features principais visualmente, com ícone + texto curto

#### Layout — opção A (4-cantos)
```
┌─────────────────────────────────────┐
│  ⚡ FEATURE 1    🔋 FEATURE 2        │
│  Texto curto    Texto curto         │
│                                      │
│         [PRODUTO CENTRAL]            │
│                                      │
│  💡 FEATURE 3   📱 FEATURE 4        │
│  Texto curto   Texto curto          │
└─────────────────────────────────────┘
```

#### Layout — opção B (vertical empilhado)
```
┌─────────────────────────┐
│  ⚡ FEATURE 1            │
│  Texto explicativo      │
├─────────────────────────┤
│  🔋 FEATURE 2            │
│  Texto explicativo      │
├─────────────────────────┤
│  💡 FEATURE 3            │
│  Texto explicativo      │
├─────────────────────────┤
│  📱 FEATURE 4            │
│  Texto explicativo      │
└─────────────────────────┘
```

#### Especificações
- **Aspect ratio**: 1:1 (2.000×2.000 px)
- **Background**: branco puro OU off-white `#FAFAFA`
- **Ícones**: usar biblioteca Canva (procurar por "icon" + tema)
- **Texto headline por feature**: máximo 20 chars
- **Texto descrição por feature**: máximo 60 chars
- **Hierarquia visual**: ícone maior, headline médio, descrição pequeno

#### Briefing tipo (preencher pelo agente)
```
Layout: opção A (4 cantos)
Features:
  1. ⚡ Bivolt — funciona em 110V e 220V
  2. 🔋 Bateria 8h — uso prolongado sem cabo
  3. 💡 3 Tons de Luz — quente, neutro, frio
  4. 🔌 USB-C — recarga moderna em 2h

Paleta: dourado primário + branco
Fontes: Montserrat Bold (headline) + Open Sans (descrição)
Tamanho fonte headline: 48pt
Tamanho fonte descrição: 24pt
```

```
│   Profundidade: 12 cm            │
│   Peso: 0.6 kg                   │

```

#### Especificações
- **Aspect ratio**: 1:1
- **Background**: branco
- **Setas/réguas**: linhas finas, cor cinza médio `#666666`
- **Texto**: ao lado das setas em fonte sans-serif média
- **Box de medidas**: canto inferior, fundo cinza claro `#F5F5F5`, texto preto

#### Briefing tipo (preencher pelo agente)

```
  - Altura (vertical, lado direito): "30 cm"
  - Largura (horizontal, embaixo): "10 cm"

  - Altura: 30 cm
  - Largura: 10 cm
  - Profundidade: 12 cm
  - Peso: 0.6 kg
Fonte: Montserrat Regular 32pt (medidas) e 24pt (box)
```

### Briefing #3 — Comparativo de Estados (imagem #4)

**Objetivo**: mostrar variação do produto (3 tons, 3 cores, 3 modos)

> Geralmente: gera as 3 imagens no Higgsfield, compõe no Canva.

#### Layout
```
┌──────────┬──────────┬──────────┐
│          │          │          │
│ ESTADO 1 │ ESTADO 2 │ ESTADO 3 │
│          │          │          │
├──────────┼──────────┼──────────┤
│ TOM      │ TOM      │ TOM      │
│ QUENTE   │ NEUTRO   │ FRIO     │
│ 3000K    │ 4000K    │ 6500K    │
└──────────┴──────────┴──────────┘
```

#### Especificações
- **Aspect ratio**: 1:1 (cada panel 666×1.000 dentro do canvas 2.000×2.000)
- **Linhas divisórias**: 4px brancas
- **Labels embaixo**: fundo cinza claro, texto centralizado
- **Manter ângulo idêntico** do produto nas 3 fotos (gerar com mesmo seed Higgsfield)

#### Briefing tipo
```
Imagens base (geradas no Higgsfield):
  - higgsfield-output-1.png (luz quente)
  - higgsfield-output-2.png (luz neutra)
  - higgsfield-output-3.png (luz fria)

Layout: 3 colunas verticais iguais
Labels:
  - Coluna 1: "TOM QUENTE" / "3000K"
  - Coluna 2: "TOM NEUTRO" / "4000K"
  - Coluna 3: "TOM FRIO" / "6500K"

Fonte labels: Montserrat Bold 36pt (headline), Regular 24pt (Kelvin)
Cor labels: branco sobre fundo cinza escuro #2A2A2A
```

### Briefing #5 — Hero com Foto Real (ajuste em Canva)

**Quando**: owner tem foto real mas precisa de fundo branco perfeito

#### Processo
1. Remover fundo: usar **Background Remover** do Canva (grátis no Pro / paid feature)
   - Alternativa grátis: **remove.bg** (3 free / mês) ou **Photopea** (online grátis)
2. Adicionar fundo branco puro `#FFFFFF`
3. Centralizar produto, ocupando 85% do frame
4. Sem sombras adicionais (Amazon prefere "flutuante" sutil)
5. Exportar PNG 2.000×2.000

## 📤 Export Final (Canva → Amazon)

### Settings de export Canva
- **Tipo**: PNG (preserva qualidade) ou JPG 100% qualidade
- **DPI**: 300
- **Tamanho**: nativo (2.000×2.000)

### Antes de subir na Amazon
- [ ] Tamanho ≥ 1.000×1.000 (recomendado 2.000×2.000)
- [ ] Sem watermark do Canva (não usar elementos Pro se conta é Free)
- [ ] Cores em sRGB (Canva exporta em sRGB por padrão)
- [ ] Texto legível em mobile (50% das pessoas vêem na palma da mão)
- [ ] Hero com fundo realmente branco puro (#FFFFFF — checar com color picker)
