## 🔑 Padrões de Combinação Keyword

### Fórmula que converte (testada em Amazon BR)
```
[Categoria] + [Modelo/Diferencial] + [Material/Cor] + [Caso de uso]
```

Exemplos:
- "abajur led + sem fio recarregável + metal dourado + cabeceira"
- "luminária touch + dimerizável usb + moderna minimalista + escritório"
- "vaso decorativo + cerâmica + branca fosca + sala"

### Fórmula que NÃO converte
- Apenas adjetivos sem categoria ("dourado moderno bonito")
- Apenas marca/modelo sem categoria ("Halter Premium")
- Apenas características técnicas ("bivolt 5V dimerizável")
